# CSS

## Emment이란?
- Emmet은 HTML과 CSS의 작성의 시간을 아주 빠르게 단축 시켜주는 에디터 확장 기능으로 코딩을 아주 빠르게 할 수 있는 자동 완성 기능이다.
- 사용법은 아주 간단하며 명령어를 입력하고 키보드 tab을 누르면 된다.
- 보통 에디터에서 Emmet이란 기능/플러그인을 설치하여 사용하지만, visual studio code에서는 별다른 설정 없이 바로 사용할 수 있다.
