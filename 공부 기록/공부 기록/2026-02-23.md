# IO Stream
### IO Stram이란?
- 입출력(Input / Output) 관련 API 제공
```java
// 패키지
import java.io.*;
```

- 단방향(Stream) 구조
	- 입력용 Stream 따로
	- 출력용 Stream 따로

- **예외 처리 필수**
	- `IOException`
	- **체크 예외(일반 예외)** -> 반드시 `try-catch` 또는 `throws` 처리 필요

## 직렬화와 역직렬화
- 직렬화
	- 메모리에 생성된 객체를 파일 또는 네트워크로 출력하기 위해 필드값을 일렬로 늘어선 바이트로 변경하는 것
	  
- 역직렬화
	- 직렬화된 바이트를 객체의 필드값으로 복원하는 것
