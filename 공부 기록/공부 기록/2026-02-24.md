# Thread(스레드)
Java에서 스레드는 **하나의 프로세스 내에서 독립적으로 실행되는 작업 흐름의 단위이다.**

### 주요 용어

| 용어       | 설명                        | 핵심 포인트   |
| -------- | ------------------------- | -------- |
| 프로세스     | 실행 중인 프로그램                | 독립 메모리   |
| 스레드      | 실행 흐름 단위                  | 메모리 공유   |
| 컨텍스트 스위칭 | 작업 전환 과정                  | 오버헤드 존재  |
| 멀티태스킹    | 여러 작업이 동시에 실행되는 것처럼 보임    | 빠른 전환    |
| 멀티스레딩    | 한 프로세스 내 여러 스레드           | 공유 자원 문제 |
| 멀티프로세싱   | 여러 CPU 사용                 | 진짜 병렬 처리 |
| 멀티프로그래밍  | CPU가 놀지 않게 프로그램을 번갈아가며 실행 | 효율 향상    |

### 안전하게 스레드 종료하기
- 스레드 강제 종료 `stop()` 메소드: `deprecated` (더 이상 사용하지 않음)


#### interrupt() 메소드 이용
- 스레드가 일시 정지 상태에 있을 때, `InterruptedException` 발생

#### 데몬 스레드
- 주 스레드의 작업을 돕는 보조적인 역할을 수행하는 스레드이다.
- 주 스레드가 종료되면 데몬 스레드도 따라서 자동으로 종료된다.
