## 1. 콜백 함수의 개념
- *콜백 함수(Callback Function)**란?
    다른 함수의 매개변수(인자)로 전달되어, 그 함수의 실행 흐름 안에서 **나중에 호출되는 함수**를 말한다.
    
- 즉, “함수 실행이 끝난 후(또는 특정 시점에) 불려지는 함수”임
    
- 자바스크립트는 **함수를 값처럼 전달**할 수 있는 _일급 객체_ 언어라서 가능하다.

---

## 2. 콜백 함수의 간단한 예시

```jsx
function greeting(name) {
  console.log(`안녕하세요, ${name}님!`);
}

function processUserInput(callback) {
  const name = "KOSTA";   // 사용자 입력값이 있다고 가정
  callback(name);         // 인자로 받은 함수를 실행
}

processUserInput(greeting);

```

👉 실행 결과
```
안녕하세요, KOSTA님!
```

여기서 `greeting` 함수는 `processUserInput` 함수에 **콜백**으로 전달되었고, 내부에서 호출됨.

## 3. 콜백 함수의 사용법

- **배열 메서드**에서 자주 사용
```jsx
const numbers = [1, 2, 3, 4, 5];

// filter 메서드에 전달된 익명 함수가 콜백
const even = numbers.filter(function(num) {
  return num % 2 === 0;
});
console.log(even); // [2, 4]

```

- **이벤트 처리**
```jsx
document.getElementById("btn").addEventListener("click", function() {
  alert("버튼이 눌렸습니다!");
});
```

- **비동기 작업** (예: setTimeout)
```jsx
setTimeout(function() {
  console.log("3초 후 실행됩니다");
}, 3000);
```

## 4. 언제 콜백을 쓰는가?

- 특정 작업이 **끝난 후 실행해야 할 동작**을 전달할 때
- **비동기 작업** (네트워크 요청, 파일 읽기, 타이머 등)
- **이벤트 처리** (클릭, 입력 등 사용자 동작)
- **반복문을 대체**하는 배열 메서드(map, forEach, filter 등)

> 비동기 처리, 이벤트, 배열 고차 함수 등에서 광범위하게 쓰인다.
