2. MVC 패턴 구조 설계/Service 영역 Interface 설계
(1) 이름, 몸무게, 비밀번호를 입력받는다.
이름.txt 을 생성해, 문서 안에
몸무게: 비밀번호 or 객체를 직렬화해서 저장한다.
다시 메뉴 화면 출력

(2) 몸무게 검색
- 이름과 비밀번호를 입력 받아, 일치하면 몸무게를 출력한다.
- 비밀번호가 일치하지 않으면 오류 출력
- 비밀번호 오류는 최대 3회만 가능, 3회를 넘으면 초기 메뉴로 돌아간다.

(3) 몸무게 변경
- 이름, 비밀번호 데이터가 일치하면, 기존 몸무게를 보여준 후
변경할 몸무게를 입력받는다.
- 파일을 열어 입력받은 몸무게를 수정한다.

4. 비밀번호 변경
- 이름과 비밀번호를 입력받아, 기존 비밀번호를 출력한 후
새로운 비밀번호를 입력받는다.
- 파일을 열어 새로운 비밀번호로 수정한다.
